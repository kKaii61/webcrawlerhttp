# This Project is for personal use

Web Crawler Nodejs - A program to crawl through every link within the page of the website using JavaScript

# How to use this project

clone this project anywhere on your computer that you refer

```
git clone https://github.com/kKaii61/webcrawlerhttp.git
```

Then open the terminal and install all the dependencies

```
npm install
```

Now find your website that you like and start the program

```
npm start https://example.com
```
